This dataset contains all 238,764,310 {{KNOWL('ec.q', 'elliptic curves')}} over $\Q$ with {{ KNOWL('ec.q.naive_height', 'naive height') }} up to $2.7 \cdot 10^{10}$ and is described in Balakrishnan-Ho-Kaplan-Spicer-Stein-Watkins [{{ LINK_EXT('MR:3540965', 'https://mathscinet.ams.org/mathscinet/article?mr=3540965') }}].

File and data format

The data is stored in 29 different text files. Each file contains all of the elliptic curves whose minimal short Weierstrass form $y^2 = x^3 + A x + B$ is a certain naive height range.

Each line corresponds to an elliptic curve, with columns separated by | characters and format given as follows. If a column has not been computed it is saved as the empty string.

Column(s) Description Example
$H$ {{KNOWL('ec.q.naive_height', 'Naive height')}} $19003704300$
$a_1$, $a_2$, $a_3$, $a_4$, $a_6$ $a$-invariants for the {{KNOWL('ec.q.minimal_weierstrass_equation', 'reduced minimal model')}} $1$, $-1$, $0$, $-102$, $-389$
$p_1$, $p_2$ Parameters to form the elliptic curve, which is isomorphic to $y^2 = x^3 + p_1 x + p_2$ $-1635$, $-26530$
$N$ {{KNOWL('ec.q.conductor', 'Conductor')}} $5940675$
$\operatorname{tam}$ {{KNOWL('ec.tamagawa_number', 'Tamagawa product')}} $1$
$n_1$, $n_2$ structure of {{KNOWL('ec.q.torsion_subgroup', 'torsion subgroup')}}, $\Z/n_1 \times \Z/n_2$ where $n_1 \in {1, 2}$ and $n_1 \mid n_2$ $1$, $1$
$\operatorname{sel}_2$ $2$-rank of the {{KNOWL('ag.selmer_group', '$2$-Selmer group')}} $2$
$w$ Root number (sign of functional equation) $-1$
$r$ {{KNOWL('ec.rank', 'rank')}} $4$
$\operatorname{CM}$ 1 if $E$ has {{KNOWL('ec.complex_multiplication', 'CM')}}, 0 otherwise $0$
$\operatorname{mwrank}_{\mathrm{lb}}$ Lower bound returned by mwrank in Sage
$\operatorname{mwrank}_{\mathrm{lb}}, \operatorname{mwrank}_{\mathrm{ub}}$ Lower and upper bounds returned by mwrank in Sage $2, 4$
$\operatorname{magma}, \operatorname{magma}_{\mathrm{lb}}, \operatorname{magma}_{\mathrm{ub}}$ Exact, lower and upper bounds on rank computed by Magma's MordellWeilShaInformation (almost always null)
$r_{\mathrm{an},0}, \delta$ Running analytic_rank_upper_bound in Sage with parameter $\delta$ returns $r_{\mathrm{an},0}$, which is an upper bound for the rank
Other columns that were used to compute rank (but probably not useful for the average user): mwrank_lb, mwrank_ub = lower and upper bounds returned by mwrank in Sage mwrank_time = amount of time mwrank needed magma, magma_lb, magma_ub = lower and upper bounds, or actual rank, returned by Magma's MordellWeilShaInformation or other functions (almost always null, because Magma was not used unless needed) r_an_0, Delta: running analytic_rank_upper_bound in Sage with parameter Delta returns r_an_0, which is an upper bound for the rank (sometimes null)
